# JobNest 

Multi-User Job Application Tracking System using PHP, AJAX, SQLite & Bootstrap

## Features

* User Registration & Login Authentication
* Session Management
* Add / Update / Delete Applications
* User-Specific Job Tracking
* AJAX-Based Dynamic Updates
* Responsive Bootstrap UI
* SQLite Database Integration
* Application Timestamp Tracking

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* jQuery
* AJAX
* PHP
* SQLite

## How To Run

```bash
php -S localhost:8000
```

Open:

```text
http://localhost:8000/login.php
```

## Project Workflow

Register → Login → Dashboard → Add Applications → View / Update / Delete Applications
